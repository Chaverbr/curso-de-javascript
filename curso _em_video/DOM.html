selecionando elementos pela arvore

modos de selecionar:

  • por marca
  • por ID
  • Por nome
  • por classe
  • por seletor
    •

selecionando por MARCA

o comando é: getElementsByTagName()

selecionando paragrafos:

var p1 = window.document.getElementsByTagName("p")[0] window.document.write(p1.innerText)

innerText: quando for só paragrafos

innerHtml: quando for o código inteiro, incluindo negrito,itálicos, quebras de linhas e etc

selecionando por ID

getElementsById(), é o comando

selecionando por NOME

getElementsByName(), é o comando

Oi eu sou o exemplo "name"

selecionando por CLASSES

getElementsByClassName(), é o comando

Oi eu sou o exemplo "classe"

selecionando por seletor

queryselector() or queryselectorall()

sinais: # para "ID" e . para "CLASS"

EVENTOS DOM

eventos com mouse:

  • mouseenter
  • mousemove
  • mousedown
  • mouseup
  • click
  • mouseout

funções: conjunto de códigos que serão executados quando um evento ocorrer

blocos: são identificados pelas 'chaves', e são o meio que utilizaremos para criar as funções
